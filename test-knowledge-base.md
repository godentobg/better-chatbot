# Knowledge Base Test Document

This is a test document for the knowledge base feature.

## Important Information

- The system supports file uploads
- Users can create multiple knowledge bases
- Content is automatically extracted and searchable
- The chat system will use this information to answer questions

## Technical Details

The knowledge base feature includes:
1. Database schema for storing files and metadata
2. API endpoints for CRUD operations
3. File upload with text extraction
4. Semantic search integration
5. Automatic context injection in chat

## Test Scenarios

When a user asks "How does the knowledge base work?", the system should be able to reference this document and provide accurate information based on the content above.
